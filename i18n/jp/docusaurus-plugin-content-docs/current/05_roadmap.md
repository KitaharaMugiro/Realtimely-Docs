# Road Maps

RealtimelyはProductionで使えるようにするために多くの課題を残しており、短期的にはその解決に注力します。
その部分の解決ができたら、
* バックエンドをユーザが保有するAWSクラウドで構築できるオプションを実装
* もしくは、さらに汎用的に利用できるような機能開発
を行っていく予定です。
これについては需要に応じてどちらを優先するか検討しようと思っています。
